Проведена приоритизация гипотез по фреймворкам ICE и RICE. Затем проанализированы результаты A/B-теста, построены графики кумулятивной выручки, среднего чека, конверсии по группам, а затем посчитаны статистическая значимость различий конверсий и средних чеков по сырым и очищенным данным. На основании анализа мной было принято решение о победе группы В.
